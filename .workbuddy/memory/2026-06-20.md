## 全局小鱼 Loading 动画
- 需求：小程序请求后端接口等待返回时，显示"一只小鱼在转圈"的加载动画
- 实现：
  - 新增 `components/fish-loading/` 组件，使用 SVG 小鱼图标 + CSS 旋转动画
  - 新增 `utils/loading.ts` 订阅式 loading 状态管理（计数器，支持多请求并发）
  - 修改 `services/request.ts`：`request()` 默认显示 loading，`requestQuiet()` 用于静默请求（如登录刷新）
  - 在 `app.json` 中全局注册 `fish-loading` 组件
  - 为所有 21 个页面 `index.wxml` 添加 `<fish-loading />` 组件
- 验证：TypeScript 编译无错误
- 涉及文件：`components/fish-loading/`、`utils/loading.ts`、`services/request.ts`、`app.json`、所有 `pages/*/index.wxml`、`images/fish-loading.svg`
